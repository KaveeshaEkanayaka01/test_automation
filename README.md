# Playwright Test Automation – Singlish Transliteration Testing

## 📌 Project Overview
This project automates testing of a Singlish-to-Sinhala transliteration system using Playwright.  
It validates how accurately the system converts informal Sinhala typing (Singlish) into correct Sinhala output.

---

## 🛠 Tools Used
- Python 3.11/3.12
- Playwright
- OpenPyXL
- Excel (.xlsx)

---

## 📂 Project Structure
- test_automation.py → Main automation script
- Assignment 1 - Test cases.xlsx → Test data file
- README.md → Project documentation

---

## ⚙️ Setup Instructions

### 1. Install dependencies
pip install -U pip
pip install playwright openpyxl
playwright install

---

### 2. Run the automation script
python test_automation.py --excel "test_automation/Assignment 1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator" --wait-ms 5000 --type-delay-ms 80 --slow-mo-ms 200 --save-every 1 --keep-open

---

## 📊 Output
- Actual Output is automatically generated by the script
- Status (Pass/Fail) is updated in Excel
- Results are saved back into the same Excel file
